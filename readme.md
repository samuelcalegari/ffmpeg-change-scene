# FFMpeg-change-scene #

Scene Change Detection with **Node.js** and **FFmpeg**

## Installation ##

> $ npm install

## Requirements ##

**FFmpeg** must be installed on your system

## Usage ##

> $ npm start -- --file=[file] --fv=[fv]
>
>file : video file name
>
>fv : filter detection value (float between 0 and 1)
